# Artificial neural networks

## Структура проекта
- lab_1
    - Нейросеть для распознавания одной цифры. В проекте содержатся шаблоны цифр в файле __number_train.xlsx__, на которых нейросеть будет обучаться. Их размер 5x7 пикселей.
    - В файле __perceptron.py__ содержится класс персептрона, который считывает данные и обрабатывает их.
    - Файл __func.py__ содержит вспомогательные функции, которые нужны для работы персептрона, например: активационная функция, инициализация весов и другие.
- lab_2
    - Данная работа повторяет __lab_1__, но после обучения передается на цифра с шумом, и нейросеть должна определить цифру и вывести график вероятности ошибки.
    - Файлы __number_h.xlsx__ и __number_weight.xlsx__ содержат данные сдвига и весов соответственно.

## Алгоритм обучения
1. Считываем входные данные из файла в виде одномерного массива нулей (белый пиксель) и единиц (черный пиксель). Так как размер изображения 5x7, то в массиве будет 35 элементов.
2. Нейросеть обучается с учителем, поэтому в файле __number_target.xlsx__ содержится единичная матрица
3. Считываем веса из файла __number_weight.xlsx__, если его нет, то иниализируем их с помощью random
4. Перемножаем входные данные с соответствующим каждому весом и складываем их. В результате получается новый массив __z__
5. Активационная функция: Сравниваем каждый элемент массива __z__ с некоторым пороговым значением. Если он больше порога, заменяем его на 1, если меньше - 0
6. Расчет ошибки сети: Вычисляем ошибку как разность между целевым и выходным значениями. На выходе получаем массив из 10 элементов (ошибок)
7. Перерасчет весов: Вычисляем новые веса по формуле `w[i][j] = w[i][j] + error[i] * x[j] * n`, где n - темп обучения
8. Записываем новые веса в файл
9. Повторяем шаги 3-8 на каждой эпохе
